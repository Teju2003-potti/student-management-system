# Student Management System

Full-stack CRUD application built using Spring Boot and MySQL.

## Features
- Add Student
- Update Student
- Delete Student
- Pagination API
- REST architecture

## Tech Stack
- Java
- Spring Boot
- MySQL
- HTML
- Bootstrap
- JavaScript

## API Endpoints

GET /students  
POST /students  
PUT /students/{id}  
DELETE /students/{id}
